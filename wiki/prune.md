### Очистка неиспользуемых ресурсов Docker

1. Открыть crontab для редактирования:
```
crontab -e
```

2. Добавить задание:
```
0 */2 * * * /usr/bin/docker system prune -af --filter "until=1h" >/dev/null 2>&1
```

3. Посмотреть список активных задач:
```
crontab -l
```
